Robotic Servo Arm → Dual-Arm Manipulator

Spring–Summer 2025
Began with a single servo-driven manipulator arm controlled by an Arduino Uno, later expanded into a dual-arm robotic manipulator with modular grippers and ESP32 networking (ESP-NOW).

  • Modeled in Onshape, sliced in OrcaSlicer, fabricated via 3D printing.
  • Upgraded control system from Arduino Uno → ESP32 for real-time networking.
  • Implemented joystick + servo motor control for independent or synchronized arm motion.
  • Tested load capacity, positional feedback, and dual-arm coordination.

Real-Time Fruit Detection (MATLAB)

Spring 2025 • 3-hour project sprint
Designed and implemented a MATLAB-based tool to detect banana-like objects in real time using color segmentation and shape analysis.

  • Yellow hue segmentation + curvature analysis for banana identification.
  • Added bounding box overlays and contour highlighting for confirmation.
  • Balanced detection thresholds for speed and accuracy (60 fps real-time performance).
  • Achieved ~90% detection accuracy across 30 test images and videos.
